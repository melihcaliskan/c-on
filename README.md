## Prerequisites

- npm >=6
- node >=14

## Installation

```sh
git clone https://github.com/melihcaliskan/c-on
cd c-on
npm install
```

## Start

```sh
npm run start:dev
or
npm run dev:up (Start with Docker Compose)
```
<h5>This will start both the web server and the mock server.</h5>
